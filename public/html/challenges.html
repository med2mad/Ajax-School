Filtering

Filter only 3 rows with age of 40.
Hint : use the "Per page" feature
Filter out the row named "AjAx".
Problem : in PostgresSql, filtering is case sensitive, so your filter characters should match the case.
Filter all rows with names starting with "Avalon".
Hint : use the "%" character in the filter to replace any string.
Filter all rows with names ending with "Avalon" (using "%").
Filter all rows that contains "el" in 1 page.
Hint : use "%" in both sides, also use the "Per page" feature.
Filter all rows with names "Lamar1" and "Lamar2" without using the "%" character.
Hint : use the "_" character in the filter to filter any single character.
Filter all rows with names with only 10 characters.
Filter out only the row with the name 66%.
Problem : Mysql and PostgresSQL use "%" with the LIKE operator, so you need to escape it (using "\").
Filter out only the row with the name 66_.
Problem : Mysql and PostgresSQL use "_" with the LIKE operator, so you need to escape it (using "\").
Filter rows with names that contain 2 specific strings, skipping the first 5 rows.
Hint : use the "%" character with the "Per page" feature and the pagination.
Update a row's name by adding multiple spaces to the start, then filter it alone using 1 "%" to filter all the space characters.
